  <action>
Create `scripts/claudux.sh` with the following implementation:

1. **Shebang and directory resolution:**
   ```bash
   #!/usr/bin/env bash
   CURRENT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
   ```

2. **Source dependencies (once per invocation):**
   ```bash
   source "$CURRENT_DIR/helpers.sh"
   source "$CURRENT_DIR/cache.sh"
   source "$CURRENT_DIR/render.sh"
   ```

3. **Background refresh logic (before rendering):**
   - Call `is_cache_stale` — if cache is stale (returns 0):
     - Get cache dir via `get_cache_dir`
     - Check PID file at `$cache_dir/fetch.pid`
     - If PID file doesn't exist OR the PID in it is not alive (`kill -0` fails):
       - Spawn background fetch: `tmux run-shell -b "echo \$\$ > '${pid_file}' && '${CURRENT_DIR}/fetch.sh'; rm -f '${pid_file}'"`
     - The `tmux run-shell -b` command writes its own PID, runs fetch.sh, then cleans up PID file
   - This is non-blocking — rendering proceeds immediately with whatever cache exists

4. **Segment routing via case statement:**
   ```bash
   case "${1:-}" in
       weekly)  render_weekly ;;
       monthly) render_monthly ;;
       sonnet)  render_model_sonnet ;;
       opus)    render_model_opus ;;
       reset)   render_reset ;;
       email)   render_email ;;
       status)
           # Combine error + stale indicators
           err="$(render_error)"
           stale="$(render_stale_indicator)"
           output="${err}${stale}"
           [[ -n "$output" ]] && printf '%s' "$output"
           ;;
       *)
           # Unknown segment — silent fail (don't pollute status bar)
           ;;
   esac
   ```

5. **Make executable:** `chmod +x scripts/claudux.sh`

IMPORTANT NOTES:
- Do NOT use `local` keyword outside of functions (this script runs at top-level, not inside a function)
- Use plain variable names instead of `local` for top-level variables
- The `status` case is special: it combines both error and stale indicators into one output
- Unknown segments produce no output (not even to stderr) to avoid polluting the status bar
