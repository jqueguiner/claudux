**Test structure:**
```bash
#!/usr/bin/env bash
# test_config.sh — End-to-end tests for @claudux_* configuration options
# Tests CONF-01 (toggles), CONF-02 (thresholds), CONF-03 (refresh), CONF-05 (bar_length)
# Requires: tmux server running (tests read/write tmux options)

set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
source "$SCRIPT_DIR/scripts/helpers.sh"
source "$SCRIPT_DIR/scripts/cache.sh"
source "$SCRIPT_DIR/scripts/render.sh"

PASS=0
FAIL=0

pass() { ((PASS++)); printf '  \033[32m✓\033[0m %s\n' "$1"; }
fail() { ((FAIL++)); printf '  \033[31m✗\033[0m %s\n' "$1"; }
assert_empty() { [[ -z "$1" ]] && pass "$2" || fail "$2 (got: '$1')"; }
assert_not_empty() { [[ -n "$1" ]] && pass "$2" || fail "$2 (got empty)"; }
assert_contains() { [[ "$1" == *"$2"* ]] && pass "$3" || fail "$3 (expected '$2' in '$1')"; }
```

**Test sections to implement:**

1. **Setup:** Write a test cache file with known data so render functions have something to work with:
```bash
setup_test_cache() {
    local cache_dir
    cache_dir="$(get_cache_dir)"
    cache_write '{
        "weekly": {"used": 50, "limit": 100, "reset_at": '$(($(date +%s) + 7200))'},
        "monthly": {"used": 200, "limit": 1000, "reset_at": '$(($(date +%s) + 86400))'},
        "models": {
            "sonnet": {"used": 30, "limit": 100},
            "opus": {"used": 10, "limit": 50}
        },
        "account": {"email": "test@example.com"},
        "error": null
    }'
}
```

2. **CONF-01 toggle tests** (6 tests):
- Set `@claudux_show_weekly off` -> call dispatcher with "weekly" -> assert empty output
- Set `@claudux_show_weekly on` -> call dispatcher with "weekly" -> assert non-empty output
- Set `@claudux_show_model off` -> call dispatcher with "sonnet" -> assert empty output
- Set `@claudux_show_model off` -> call dispatcher with "opus" -> assert empty output
- Set `@claudux_show_email on` -> call dispatcher with "email" -> assert non-empty output
- Set `@claudux_show_email off` -> call dispatcher with "email" -> assert empty output

For each toggle test:
```bash
set_tmux_option "@claudux_show_weekly" "off"
output=$("$SCRIPT_DIR/scripts/claudux.sh" weekly)
assert_empty "$output" "CONF-01: weekly hidden when show_weekly=off"

set_tmux_option "@claudux_show_weekly" "on"
output=$("$SCRIPT_DIR/scripts/claudux.sh" weekly)
assert_not_empty "$output" "CONF-01: weekly shown when show_weekly=on"
```

3. **CONF-02 threshold tests** (2 tests):
- Set `@claudux_warning_threshold 90` and `@claudux_critical_threshold 95` -> render bar at 50% -> assert green color (colour34)
- Set `@claudux_warning_threshold 30` and `@claudux_critical_threshold 40` -> render bar at 50% -> assert red color (colour196)

Call `render_bar 50 10` directly and check for color codes in output.

4. **CONF-03 refresh interval test** (1 test):
- Set `@claudux_refresh_interval 1` -> write cache -> sleep 2 -> assert `is_cache_stale` returns 0 (true)
- Set `@claudux_refresh_interval 9999` -> assert `is_cache_stale` returns 1 (false, cache is fresh)

5. **CONF-05 bar length tests** (3 tests):
- Call `render_bar 50 10` -> count filled+empty characters -> assert 10 total
- Call `render_bar 50 20` -> count filled+empty characters -> assert 20 total
- Call `render_bar 50 3` -> count filled+empty characters -> assert 5 (clamped from 3)

Count bar characters by stripping tmux formatting and counting Unicode block chars.

6. **Teardown:** Reset all tmux options to defaults after tests:
```bash
teardown() {
    set_tmux_option "@claudux_show_weekly" "$CLAUDUX_DEFAULT_SHOW_WEEKLY"
    set_tmux_option "@claudux_show_monthly" "$CLAUDUX_DEFAULT_SHOW_MONTHLY"
    set_tmux_option "@claudux_show_model" "$CLAUDUX_DEFAULT_SHOW_MODEL"
    set_tmux_option "@claudux_show_reset" "$CLAUDUX_DEFAULT_SHOW_RESET"
    set_tmux_option "@claudux_show_email" "$CLAUDUX_DEFAULT_SHOW_EMAIL"
    set_tmux_option "@claudux_warning_threshold" "$CLAUDUX_DEFAULT_WARNING_THRESHOLD"
    set_tmux_option "@claudux_critical_threshold" "$CLAUDUX_DEFAULT_CRITICAL_THRESHOLD"
    set_tmux_option "@claudux_refresh_interval" "$CLAUDUX_DEFAULT_REFRESH_INTERVAL"
    set_tmux_option "@claudux_bar_length" "$CLAUDUX_DEFAULT_BAR_LENGTH"
}
```

7. **Summary output:**
```bash
echo ""
echo "Results: $PASS passed, $FAIL failed"
[[ $FAIL -eq 0 ]] && exit 0 || exit 1
```

Make the script executable: `chmod +x tests/test_config.sh`
